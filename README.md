# 5XO

5XO is a web application developed in Python using Flask.

The game is inspired by Tic-Tac-Toe but is played on a 50x50 board.
The objective is to place five consecutive symbols (X or O)
horizontally, vertically or diagonally.

## Technologies

- Python 3
- Flask
- HTML
- CSS
- JavaScript

## Features

- 50x50 board
- Two-player game
- Five in a row wins
- Restart game
- REST API

## Run

Create virtual environment

```bash
python3 -m venv venv
```

Activate

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

Open

```
http://localhost:5000
```
